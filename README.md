# ID Digital Local PRO

Versión PRO de la app de gestión de identidad digital y contactos.

## Funcionalidades avanzadas
- Exportar e importar contactos (CSV, vCard)
- Sincronización en la nube
- Personalización avanzada de la tarjeta
- Escaneo de QR para agregar contactos
- Notas privadas y campos personalizados
- Seguridad extra (PIN/biometría)
- Estadísticas de uso
- Sin publicidad
- Multi-idioma
